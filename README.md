# 🎉 Acerca del proyecto

**Party Rentals Management System** es una plataforma orientada a empresas de organización de eventos y renta de artículos para fiestas, diseñada para optimizar reservaciones, inventario, pagos y administración operativa desde una sola interfaz.

La plataforma fue desarrollada para:

- 🎊 Gestionar eventos y reservaciones
- 📦 Administrar inventario y stock
- 👥 Gestionar clientes y usuarios
- 💳 Controlar pagos y contratos
- 📅 Organizar calendarios de eventos
- 📊 Generar reportes administrativos
- ⚡ Automatizar procesos empresariales

El sistema integra herramientas modernas para mejorar la productividad y el control operativo de negocios dedicados a eventos y entretenimiento.

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/usuario/party-rentals-management-system.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd PartyRentalsManagementSystem
```

---

## 3️⃣ Instalar dependencias

```bash
npm install
```

---

## 4️⃣ Configurar base de datos

```env
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
```

---

## 5️⃣ Ejecutar servidor

```bash
npm run dev
```

---

## 6️⃣ Abrir aplicación

```bash
http://localhost:3000
```
